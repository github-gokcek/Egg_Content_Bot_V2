# Egg Content Bot V2

Discord.js ile oluşturulmuş bir Discord botu.

## Kurulum

1. Bağımlılıkları yükle:
```bash
npm install
```

2. `.env` dosyasını oluştur ve bot token'ını ekle:
```
DISCORD_TOKEN=your_bot_token_here
DISCORD_CLIENT_ID=1461056941567770684
DISCORD_PUBLIC_KEY=b54a9cde6028e7b3ab22ad955c00cd9e993fbb26692897883ecf079af98a36c0
```

3. Botu başlat:
```bash
node index.js
```

## Komutlar

- `!ping` - Bot latency'sini gösterir
- `!hello` - Selamlaşır
